# MVP
# **Introdução e objetivos**

Este trabalho explora os dados coletados por aproximadamente dois meses na India sobre as comorbidades que estão relacionadas as doenças crônicas dos rins(Chronic KIdney Disease dataset - ckd). O objetivo dessa análise exploratória de dados é analisar quais atributos desse dataset estão mais relacionados as doenças crônicas do rins.

---



# Descrição do problema

A doença ou insificiência renal crônica caracteriza-se quando os rins perdem
suas funções de maneira lenta e gradual. Segundo o ministério da saúde, doenças
crônicas dos rins podem ter multiplas causas e multiplos fatores de risco.
Uma coleta de dados de uma determinada população permite a tentativa de correlacionar ou associar quais são as variáveis que mais contribuem para o diagnóstico crônico das perdas das funções renais. No que tange a natureza desse dataset, o conjuto de dados apresenta no contexto de aprendizado supervisionado, com intuito posterior de desenvolver modelos priditivos para prever doenças crônicas dos rins. Entretanto, a fase atual desse trabalho limita-se apenas a análise exploratória de dados (*Exploratory data analysis - EDA*), no pré-processamento dos dados e tratamento dos missings. Portanto tal projeta permite estabelecer relações qualitativas ou quantitativas dos atributos do dataframe com os quadros crônicos renais.

---


**Hipóteses do problema**
- Quais são os atributos que mais contribuem para estado crônico de mal funcionamento dos rins;

- Há prepônderãncia de algum atributo categórico em casos positivos, relacionado a doenças crônica dos rins;

- Algum atributo de maneira isolada permite atribuir ao paciente  mal fucionamento crônico das suas funções renais.
